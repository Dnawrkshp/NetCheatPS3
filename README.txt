NetCheat PS3 
Written by Dnawrkshp

Compile:
	Written in Visual C# 2012 Express
	Make sure to add a reference to ps3tmapi_net.dll (download off the internet or get from your installation of PRoDG)

Run:
	Find the ps3tmapi.dll somewhere on the web,
		or move it from ?:\Program Files\SN Systems\PS3\bin (if you installed the PRoDG Debugger),
		and put it in the same directory as NetCheat PS3
	
Tips:
	Double click a code to toggle constant write
	Ctrl-C on result(s) will copy them into the clipboard in code format
	Search results do not automatically refresh

Please report any bugs to the NetCheat v3 ps3hax.net thread
	
Enjoy.
